# Learnt

A minimal iOS app for tracking daily learnings.

## Philosophy

> Every element must earn its place. If removing something makes the app clearer, cut it.

## Features

- **One entry per day** - Constraint creates habit
- **Voice or text input** - Capture thoughts however feels natural
- **Weekly activity dots** - See your consistency at a glance
- **Monthly summaries** - AI finds patterns in your learnings (v2+)

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Getting Started

1. Clone the repo
2. Open `Learnt.xcodeproj` in Xcode
3. Build and run (Cmd+R)
